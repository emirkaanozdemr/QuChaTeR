# QuChaTeR

## Everything made by Emir Kaan Özdemir

The official implementation and source code of "QuChaTeR: A Hybrid Quantum-Chaotic Temporal Framework for Earthquake Prediction", which has been accepted for presentation at the IEEE ICASSP 2026 conference at the Satellite Workshop on Quantum Machine Learning in Signal Processing and Artificial Intelligence.

## Repo Structure 
`./src` folder contains source code for the framework. `./train-examples` folder contains real model integration of the framework.
